## BEEFへのログイン方法
  1. https://beef.center.kobe-u.ac.jp/ へアクセスします． 
  2. 「神戸大学シングルサインオン(Knossos)用ログインページ」 の画面が出ますので、ID, パスワードを入力すると、BEEFの初期画面が出ます．
